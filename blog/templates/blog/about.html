    关于这个博客

    项目简介

    这是一个用 Django 框架构建的多用户博客系统，由计算机专业大二学生开发。 项目旨在学习Web开发的全流程，从后端到前端，从数据库设计到用户界面。

    通过这个项目，可以学习到现代Web开发的核心技术栈，包括Python后端开发、 数据库设计、RESTful API设计、前端界面开发等实用技能。

    功能特点
    多用户系统
    • 用户注册与登录
    • 个人资料管理
    • 权限控制系统
    博客功能
    • 文章发布与管理
    • 评论互动系统
    • 文章分类标签
    技术栈
    后端技术
    • Python 3.x - 主要编程语言
    • Django 4.x - Web框架
    • SQLite - 开发数据库
    • Django ORM - 数据库操作
    前端技术
    • HTML5 + CSS3 - 页面结构
    • Bootstrap 5 - 响应式框架
    • JavaScript - 交互逻辑
    • Django Template - 模板引擎
